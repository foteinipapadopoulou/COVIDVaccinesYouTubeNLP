In this folder, the code for the dataset extraction is displayed.
* `Comment.py` is the class for the representation of the extracted comments.
* `extracting_comments.py` is the module used for extracting the comments given a csv containing the video ids
* `extracting_video_ids.py` is the module used for extracting the video ids from Youtube API related to COVID-19 vaccines
* `filtering_comments.py` is the module used for drop non-English comments and out of the specific timeframe comments.
* `utils.py` is the module that contains utility functions and helper classes used in the dataset extraction process.
